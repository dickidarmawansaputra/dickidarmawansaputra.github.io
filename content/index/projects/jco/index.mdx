<p style="text-align: justify;">The JCO Indonesia mobile ordering app! Our JCO delivery service app allows customers in Indonesia to place orders via Android devices. Customers are required to register with an email and password before placing an order. The same login can be used for the application and web (jcodelivery.com) JCO delivery service. Now you can easily place your JCO orders anywhere, anytime, and see the current status of your orders. Be sure to select your location before placing your order.</p>

<b style="text-align: justify;">What i do: Integrate API from JCO, create some required functions and APIs, modify data requirements and data response. (Backend - REST API)</b>
